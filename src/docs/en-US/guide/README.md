# Introduction

- Gofi is a web disk application whose UI is built with Ant Design Pro and it is driven by Golang
- frontend is based on [Ant Design Vue Pro](https://github.com/sendya/ant-design-pro-vue)， backend is powered on Golang [iris](https://github.com/kataras/iris) framework

## Features

- Use sqlite3 as database,all static forntend assets are embedded in executateable binary file
- Based on Go,you can run Gofi on Android, except Mac/Linux/Windows
- Modern User Interface based on Ant Design Vue Pro,and support i18n

## Technology Stack

- [Ant Design Vue Pro](https://github.com/sendya/ant-design-pro-vue)
- [iris](https://github.com/kataras/iris)

## Support environment

- Modern browsers
- MacOS / Windows / Linux / Android
